run_simulation(self) -> None
The main rescue mission loop runs here.  The simulation runs until either
all the hostages have been rescued, or there are no more soldiers left
to rescue hostages.  The steps to follow are:
 
1. Print the start up message, "Get to the choppa!"
2. While the simulation is still going:
        3. Print the current statistics
        4. Deploy the next soldier into the enemy base to
            attempt rescuing the next hostage.
        5. If the hostage was not rescued goto step 2.
        6. Otherwise do steps 7-13.
        7. Print a message "{hostage} rescued from enemy
            base by soldier {solder}".
        8. Roll the die, 1-100, to get the predators chance to
            defeat the soldier and display the message, "{soldier} encounters the
            predator who rolls a #".
        9. If the die roll is greater than the predator's
            chance to defeat the soldier, declare victory of the soldier over the predator.
            Have the hostage board the chopper,
            add the soldier back to the end of the bunker, and go to step 2.
        10. Otherwise declare victory of the predator over the
            soldier and continue to step 11.
        11. Roll the die again, 1-100, to get the predators
            chance to defeat a hostage and display the message, "{hostage} encounters the
            predator who rolls a #".
        12. If the die roll is greater than the predator's
            chance to defeat the hostage, declare victory of the hostage over the predator,
            then have the hostage board the chopper.
            Go to step 2.
        13. Otherwise declare victory of the predator over the
            hostage and go to step 2.
 
14. At the end of the simulation any soldier that may be left in the bunker
should board the chopper.
15. If there are any remaining passengers on the chopper, rescue them to safety.
16. Print the statistics one last time.
@return: None

main() -> None
Run the rescue mission simulation. The simulation requires four command-line arguments:
1. the seed for the random number generator (positive integer)
2. the number of hostages (positive integer)
3. the number of soldiers (positive integer)
4. the number of guerrillas (positive integer)
 
If all arguments are supplied, it will create the rescue mission and run the simulation.
